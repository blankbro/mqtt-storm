# MQTT-Storm: MQTT Stress Testing Tool

像风暴一样冲击和评估MQTT服务器的性能

## Build

```shell
$ git clone https://github.com/timeway/mqtt-storm.git
$ cd mqtt-storm/cmd
$ ./build.sh
```

## Run

```shell
$ ./start.sh
```

see [storm.http](cmd/storm.http)
